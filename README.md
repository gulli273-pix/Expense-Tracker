# Expense-Tracker
Programming 2 Final Project


# Project description
A simple, free program that tracks your money, habits and history.


# Features
- Add Transactions
  - Income
  - Expense
- View Transaction History
- View Total Transaction Summary + Remaining Balance
- View Spending Habits
- Edit Past Transactions


# Specs
- Language: Java (Version 11 or higher)
- Any IDE will work (I used VS Code)


# How to run
Option 1:
- On main page click <> Code
- Click download zip
- Extract the files to your prefered folder
- In your preferred IDE open this folder
- Run the code as normal

Option 2:
- On main page click <> Code
- Click create codespace
- Run from this
(this is my preferred method, easy and no extra folders)
